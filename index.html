VerSearch

VerSearch allows for an easy search of any bible verse.

biblesearch uses bible-api.com to grab verses in the

World English Bible translation!

How To Use The Tool:

First, select the book of the Bible you are looking up.

Then, search up the verse.

As an example, if you wanted to search up "John 3:16"

you would select "John" as the book, then type "3:16" in the verse and chapter.

If you want want to search up an entire chapter, you can just type the chapter (i.e. "John 3")
